\# Palindrome Checker App



\## Overview

Console-based Java application that validates palindrome using multiple approaches.



\## Implemented Use Cases



\### UC1 – Application Entry

Displays app name and version.



\### UC2 – Hardcoded String

Checks a predefined string.



\### UC3 – Reverse Using Loop

Uses string concatenation.



\### UC4 – Two Pointer Technique

Uses char\[] and index comparison.



\### UC5 – Stack (LIFO)

Uses Stack to reverse string.



\### UC6 – Queue vs Stack

Demonstrates FIFO vs LIFO behavior.



\### UC7 – Deque

Uses double-ended queue for optimized comparison.



\## Data Structures Used

\- String

\- char\[]

\- Stack

\- Queue

\- Deque



\## Time Complexity

All approaches run in O(n).
